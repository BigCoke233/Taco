# Taco

A Next.js bloging program.

## Development

```
pnpm install
bun run dev
```

## Technology Stack

* **Major Library:** React
* **Framework:** Next.js
* **Bundler:** ~~Webpack~~ Turbopack
* **Markdown Parser:** Markson
* **HTML Loader:** Interweave
* **Comment System:** Waline, Giscus
* **User Interface:** NextUI, TailwindCSS, Framer Motion
